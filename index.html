Git

Algumas etapas para utilizar o Git

Configuração

  • Download Git aqui
  • Instalar git
  • Criar conta no github

Utilização

A manutenção dos repositórios pode ser feito através do GitHub Desktop, porém para um maior controle do versionamento dos repositórios, é aconselhável usar o Gitbash, que é uma ferramenta de linha de comando. O Gitbash é fácil de ser utilizado e tem um sistema de ajuda bem amigável.

Criando Primeiro Repositório

  • Acesse sua conta do GitHub, vá no menu "es;Create new..."es; e selecione "es;New repository"es;
  • Copie o endereço do seu repositório
  • Abra o Gitbash e utilize git clone e cole o endereço copiado no GitHub

Está pronto seu repositório, no GitHub e no seu computador. Agora é só desenvolver e adicionar as atualizações, fazer os commits e pushs para o gitHub. Mais informações podem ser obtidas aqui
